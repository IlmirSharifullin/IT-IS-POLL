# IT-IS-POLL
Сервис для создания и проведения опросов и последующего анализа результатов, предоставления статистики.

# Запуск проекта
1) docker compose up --build -d
2) docker exec -it <container_id> bash
3) cd project_name
4) python3 manage.py migrate
5) python3 manage.py createsuperuser

# Фичи проекта
1) Предоставление анализа ответов пользователей: статистика по возрасту, графики распределения ответов, вордклауды
2) Возможность экспорта ответов в excel/pdf
3) Новостной блог встроенный в основную страницу опросника для удержания пользователей + возможность комментирования новостей и оставления реакций
